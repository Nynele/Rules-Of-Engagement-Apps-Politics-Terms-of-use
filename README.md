# Rules of Engagement: The Grey State — Custom Bots Webpage

This repository hosts the static landing page, Terms of Service, and Privacy Policy for the custom Discord bots integrated into the **Rules of Engagement: The Grey State** ecosystem.

## Purpose

To comply with Discord Developer Portal requirements, all active or verified application profiles must provide:
* A public Terms of Service URL
* A public Privacy Policy URL
* An installation/invite link (which is kept disabled here, as these custom bots are private system utilities)

This repository provides a minimalist, dark-themed, and generic set of pages that satisfies these requirements without exposing internal bot mechanisms.

## Repository Structure

* **`index.html`**: The main landing page listing the bots' generic description, support contact, and a styled disabled install button.
* **`terms.html`**: The generic Terms of Service page.
* **`privacy.html`**: The generic Privacy Policy page.
* **`styles.css`**: Minimalist CSS rules implementing a dark theme with Arial typography.
